# Game of Life

A lightweight, dependency-free implementation of John Conway's 1970 cellular
automaton, built with HTML, CSS, and Canvas 2D.

## Features

- Conway's original evolution rules
- Play, pause, step, randomize, and clear controls
- Generation and living-cell counters
- Adjustable simulation speed
- Adjustable grid size
- Preset specimens for well-known Life patterns
- Click-and-drag cell drawing

## Preview locally

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
